Project statistics

We've added two more statistic for the project's page. We need to show to each possible contributor how "alive" a project is, so following that train of thought, we've added two new graphics to show the acitivy inside a project:

Commits Activity
This chart shows how active a project is in a day to day basis. We don't know much about the commits by looking at the bars, but we know if there's been anything done to the code lately.

Pull Requests Activity
Even if there are no commits, the project could still be active just by merging or closing pull requests, so it's also important to know if there are contributors already working on the project and how ofter their contributions get handled (whether they're accepted and merged or just closed).
